### 问题1：页面无法访问
```bash

curl http://localhost:8080
```
**解决**：确保前端服务器在运行

### 问题2：API调用失败
```bash
# 检查后端服务器
curl http://localhost:3000/health
```
**解决**：确保后端服务器在运行

- 检查网络连接

- 按F12打开开发者工具
